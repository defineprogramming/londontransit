Introducing LondonTransit

LondonTransit brings London's essential transport network to your discord server. It provides live information on London's buses, underground, overground, DLR, Elizabeth line, trams and river services.

Features

Live departures

Get live departures for any bus stop, tube station, tram stop, river pier or DLR station in London.
